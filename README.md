# LCA-APP

Dette repository indeholder dokumentation til en Microsoft Power Apps Canvas-app, der registrerer og viser CO₂-aftryk fra en byggeplads baseret på Excel-datakilde.

## Indhold

- `docs/powerapp_design.md` – detaljeret vejledning til opsætning af appen, inklusive skærmlayout, Power Fx-formler og datakrav.

## Hurtig start

1. Formater alle ark i Excel-filen som tabeller og publicér filen på SharePoint/OneDrive.
2. Følg vejledningen i [`docs/powerapp_design.md`](docs/powerapp_design.md) for at oprette appen i Power Apps Studio.
3. Del og udgiv appen til byggepladsens brugere.
